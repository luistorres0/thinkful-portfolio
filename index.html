I'm a Software Engineer living in the Nashville area. I develop websites and web applications.

About Me

Ever since I picked up my first joystick back in the Atari 2600 days, I've been fascinated by computers. In college, I took an introductory C++ course and fell in love with coding. That's when I knew that I wanted a career in software development.

I like working in teams to create interactive, intuitive web applications using my knowledge of HTML, CSS, and Javascript. I'm also familiar with several other popular web frameworks and tools such as React, Nodejs, Bootstrap, VSCode, Git, and Github.
